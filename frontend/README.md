### Front End

ATS Nexus - AI Resume Matcher Frontend

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:3000`.

## Features

- Professional, futuristic UI with gradient backgrounds
- Two side-by-side text areas for Job Description and Resume
- Neon glow effect on the "Scan Application" button
- Glassmorphism design for the Compatibility Report card
- Smooth fade-in animations using Framer Motion
- Responsive design that works on desktop and mobile

## Building for Production

```bash
npm run build
npm start
```
